Nested Loop
A nested loop means a loop inside another loop.
üëâ Outer loop runs first
 üëâ Inner loop runs completely for each iteration of outer loop

Example 1: Simple Nested Loop
for i in range(0, 2):
    for i in range(0, 1):
        print("the value of i :", i)



Output
the value of i : 0
the value of i : 0

The inner loop runs every time the outer loop runs.
