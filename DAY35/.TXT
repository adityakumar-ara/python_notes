Program 2: Print Even Numbers Using Function



def print_even_number(start, end):
    for i in range(start, end):
        if i % 2 == 0:
            print(i)

print_even_number(10, 21)



Output:
10
12
14
16
18
20


This function prints even numbers between the given start and end values.
