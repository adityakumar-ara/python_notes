<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<-------------Function Definition--------------------->>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
A function is created using the keyword def
Syntax:
def function_name():
    # execute some task




Function Call
After defining, we must call the function to run it:
def ujwal_function():   #"ujwal_function" this is function name
    print("Hii Ujwal! how are u") #Function body performs the task.


ujwal_function() # call function

Output:
Hii Ujwal! how are u



Function with Multiple Statements
def ujwal_function():
    print("Hii Ujwal! how are u")
    print("This is Python programming language")
    print("This is Java Programming Language")
    print("C++ is Programming language")

ujwal_function()


Output:
Hii Ujwal! how are u
This is Python programming language
This is Java Programming Language
C++ is Programming language



Global & Local Variables
Example: Only Global Variable
a = 3
print(a ** 2)  # a^2 = 3^2 = 9



Output:
9



Example: Local + Function Calling Multiple Times
def print_sum_of_numbers(a, b):
    sum = a + b
    print("Hii Ravi!, here is the sum of a & b:", sum)

print_sum_of_numbers(22, 5)

print("starting")
print("running")

print_sum_of_numbers(99, 1)

my_variable = 10
print(my_variable)



Output:
Hii Ravi!, here is the sum of a & b: 27
starting
running
Hii Ravi!, here is the sum of a & b: 100
10
